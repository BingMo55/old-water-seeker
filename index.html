LOST


Ever had to buy another bottle because you lost yours and couldn't find it? WaterSeeker seeks to solve that issue by giving each bottle an ID so it can be return back to you!




REFILL


Ever had an empty bottle but didn't know where to refill it? WaterSeeker seeks to alleviate that issue by allowing students to find the closest water refill station.
